# 📊 Data Analysis on CSV Files

## 📌 Objective
Analyze sales data using Pandas and visualize insights using charts.

---

## 🛠 Tools & Libraries
- Python
- Pandas
- Matplotlib
- Jupyter Notebook / Google Colab

---

## ⚙️ What I Did

1. Loaded the CSV file using Pandas.
2. Explored the dataset using `head()`.
3. Used `groupby()` to analyze:
   - Total sales by product
   - Total sales by region
4. Used `sum()` to calculate total values.
5. Created a bar chart using `plot()` to visualize sales.

---

## 📂 Dataset Format (Example)

| Product | Region | Sales |
|--------|--------|-------|
| A      | East   | 100   |
| B      | West   | 200   |

---

- Bar chart showing sales distribution
